# Minimalist Form with Flexbox and Icons

## Description

A minimalist form created using HTML, CSS (Flexbox), and Font Awesome icons. This form includes inputs for basic user information and implements a clean design with subtle hover and focus effects.

## Features

- Minimalist design with clean and modern UI
- Flexbox layout for form alignment
- Font Awesome icons for input fields
- Responsive and accessible
- Subtle hover and focus effects for interactivity
